Introduction

AutoTuner is a program that can tune human voice to sing a simple melody. It is built in MATLAB.

We use following audio processing methods to to achieve this function:
- Pitch tracking: analyze input human voice and melody, obtaining their time and frequency information.
- Beat tracking: detect onset and match voice with input melody.
- Phase vocoder: shift pitch.
